# Pokecore

Welcome to Pokecore, the ultimate competitive Pokémon repository! Built with Angular and Firebase, this application offers a robust platform for Pokémon trainers to analyze, plan, and enhance their competitive strategies. Using HTML, CSS, TypeScript, and Bootstrap, Pokecore provides a user-friendly and responsive interface.
